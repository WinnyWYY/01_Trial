# 刷题系统

这是一个基于 PDF 提取和 Web 界面的刷题系统，支持单选题、多选题和是非题。

## 系统组成

### 1. PDF 题目提取器 (`pdf_extractor.py`)

从 PDF 文件中自动提取题目，识别红色字体的正确答案。

**使用方法：**

```bash
python pdf_extractor.py <pdf 文件路径> [输出 json 路径]

# 示例
python pdf_extractor.py questions.pdf questions.json
```

**功能特点：**
- 自动识别题目编号（如 "1.", "2." 等）
- 识别选项（A. B. C. D. 等）
- 通过颜色识别正确答案（红色字体）
- 自动判断题型（单选、多选、是非题）
- 导出为标准 JSON 格式

### 2. 刷题系统 Web 界面 (`index.html`)

一个美观的响应式 Web 刷题应用。

**使用方法：**
1. 在浏览器中打开 `index.html`
2. 上传由提取器生成的 JSON 题目文件
3. 开始刷题！

**功能特点：**
- 📊 实时统计：总题数、已答题数、正确数、正确率
- 🎯 题型筛选：按单选、多选、是非题筛选
- ⌨️ 键盘导航：左右箭头切换题目，Enter 提交答案
- 📱 响应式设计：支持手机、平板、电脑
- 🎨 美观界面：渐变色设计，流畅动画
- ✅ 即时反馈：答题后立即显示正确/错误
- 💾 本地运行：无需服务器，直接在浏览器中使用

### 3. 示例题目 (`sample_questions.json`)

包含 5 道示例题目（单选、多选、是非题），用于测试系统。

## 使用流程

1. **准备 PDF**：确保你的 PDF 题目文件中，正确答案用红色字体标注
2. **提取题目**：运行 `python pdf_extractor.py your_file.pdf`
3. **开始刷题**：在浏览器打开 `index.html`，上传生成的 JSON 文件

## 技术栈

- **后端提取**：Python + PyMuPDF (fitz)
- **前端界面**：原生 HTML/CSS/JavaScript（无需任何框架）
- **数据存储**：JSON 格式

## 注意事项

1. PDF 中的正确答案必须用**红色字体**标注才能被正确识别
2. 题目格式应规范：题号 + 题干 + 选项（A. B. C. D.）
3. 是非题会识别"正确"/"错误"或"√"/"×"等标记
4. 如果提取效果不理想，可以手动编辑生成的 JSON 文件进行调整

## 自定义

你可以根据需要修改：
- `pdf_extractor.py` 中的颜色判断逻辑（`is_red_color` 函数）
- `index.html` 中的样式和交互逻辑
- JSON 数据结构以添加更多字段（如解析、难度等）
